# SuraChain 🗳️

**Censorship-resistant on-chain voting protocol** built with Solidity + Foundry.

A transparent voting system where votes are immutable and verifiable by anyone.

---

### ✨ Features
- One address = one vote (double voting prevention)
- Time-bound elections with automatic deadline enforcement
- Gas-optimized and secure
- Custom errors + events for full transparency
- Comprehensive test coverage with Foundry

### 🛠️ Tech Stack
- **Solidity** ^0.8.20
- **Foundry** (Forge, Anvil, Cast)
- **OpenZeppelin** best practices

### 🚀 Quick Start

```bash
git clone https://github.com/saftanasdalihin/sura-chain.git
cd sura-chain

# Install Foundry
curl -L https://foundry.paradigm.xyz | bash

forge build
forge test -vvv
```
